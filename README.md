# LensAI Detect - Real-Time Object Detection

LensAI Detect is a high-performance, browser-based object detection system powered by **TensorFlow.js** and the **MobileNet V2** neural network engine. It allows users to identify and track objects in real-time using their device's camera, with all processing performed locally for maximum privacy and speed.

## Key Features

- **Real-Time Detection**: Instantly identifies multiple objects with bounding boxes and labels.
- **Privacy First**: All AI inference happens directly in the browser—no data is sent to external servers.
- **Dynamic UI**: Modern, responsive interface with real-time telemetry and status updates.
- **High Efficiency**: Optimized for performance using the MobileNet V2 engine to ensure smooth operation even on lower-end devices.

## Tech Stack

- **Framework**: React.js with Vite
- **Language**: TypeScript
- **AI/ML**: TensorFlow.js (MobileNet V1/V2)
- **Styling**: Vanilla CSS with modern glassmorphism and animations
- **Icons**: Lucide React

## Getting Started

1.  Clone the repository.
2.  Install dependencies: `npm install`
3.  Run the development server: `npm run dev`
4.  Open the application in your browser and initialize the camera to start detection.
